##  建表规范

    
    /* 建表规范 */ ------------------
        -- Normal Format, NF
            - 每个表保存一个实体信息
            - 每个具有一个ID字段作为主键
            - ID主键 + 原子表
        -- 1NF, 第一范式
            字段不能再分，就满足第一范式。
        -- 2NF, 第二范式
            满足第一范式的前提下，不能出现部分依赖。
            消除符合主键就可以避免部分依赖。增加单列关键字。
        -- 3NF, 第三范式
            满足第二范式的前提下，不能出现传递依赖。
            某个字段依赖于主键，而有其他字段依赖于该字段。这就是传递依赖。
            将一个实体信息的数据放在一个表内实现。
