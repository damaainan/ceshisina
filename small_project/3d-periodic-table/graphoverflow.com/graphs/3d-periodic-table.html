Periodic Table of Elements

This is a Bohr model of the atom. This model was proposed by Niels Bohr in 1913. It is not completely correct, but has many features that are sufficient for a general understanding. In this representation, the electrons are rotating in a 2D plane unlike the Bohr model which suggests the electrons move in a 3D path around the nucleus. The 2D representation is for simplicity>

A 3d visualization of periodic table. This 3d representation has a table view which shows initially and an atomic view (click any elements "Explore atom" button).
